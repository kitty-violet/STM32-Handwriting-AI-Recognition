# 使用说明

## 1. 环境准备

### STM32 端

- Keil MDK 5。
- ARM Compiler 6。
- STM32F103VE 对应 Keil Device Pack。
- 支持触摸屏、LCD、TF 卡和 ESP8266 WiFi 模块的开发板。

### PC 端

建议使用 Python 3.10 或更新版本。进入 `02_PC端网页与训练工具/` 后安装依赖：

```powershell
pip install -r requirements.txt
```

如果只运行网页接收端，不使用训练和 PC-CNN，可以先安装基础依赖；如果要训练或使用 PC-CNN，需要安装 PyTorch。

## 2. 准备 TF 卡

将 `03_TF卡示例文件/` 内所有内容复制到 TF 卡根目录，最终结构类似：

```text
MODEL/
TEST/
Font/
WEB/
```

其中 `MODEL/FNN.BIN` 用于 STM32 端加载 FNN 权重，`TEST/` 用于批量测试，`Font/` 用于中文显示。

## 3. 修改 WiFi 配置

打开：

```text
01_STM32固件源码/User/wifi_report.h
```

修改以下占位符：

```c
#define WIFI_REPORT_SSID          "YOUR_WIFI_SSID"
#define WIFI_REPORT_PASSWORD      "YOUR_WIFI_PASSWORD"
#define WIFI_REPORT_SERVER_IP     "YOUR_PC_IP"
#define WIFI_REPORT_SERVER_PORT   "8000"
```

`YOUR_PC_IP` 应填写电脑在同一热点/局域网下的 IPv4 地址。

## 4. 编译烧录 STM32 工程

打开工程：

```text
01_STM32固件源码/Project/RVMDK（uv5）/BH-F103.uvprojx
```

使用 Keil MDK 5 + ARM Compiler 6 编译。工程已包含包内 scatter 文件 `STM32F103VE.sct`，不依赖外部绝对路径。

如果只是先体验基础功能，也可以直接烧录：

```text
01_STM32固件源码/prebuilt/Template_demo_placeholder_wifi.hex
```

注意：这个 HEX 使用占位 WiFi 配置，不能直接连接你的电脑网页端。需要 WiFi 上报、翻译、PC-CNN 等联网功能时，请先修改 `wifi_report.h` 后重新编译烧录。

## 5. 启动 PC 接收端

进入：

```powershell
cd 02_PC端网页与训练工具/网页接收端
```

常规启动：

```powershell
python web_receiver.py
```

需要 DeepSeek 网页翻译时：

```powershell
python web_receiver.py --deepseek
```

也可以双击 `启动接收端_可见窗口.bat`。启动后访问：

```text
http://127.0.0.1:8080/
```

## 6. 演示操作

### 单字符识别

1. 烧录固件并启动开发板。
2. 在触摸区域写一个数字或字母。
3. LCD 显示笔迹、缩略图和识别结果。
4. KEY2 可切换 FNN/CNN 模型，具体取决于固件当前模式支持。

### TF 卡批量测试

1. 插入准备好的 TF 卡。
2. 按 KEY1 切换到 TF 卡测试模式。
3. 串口和网页会显示 MNIST/个人测试集准确率和平均耗时。
4. 批量测试中可按 KEY1 中断。

### 字符串识别

1. 按 KEY1 切换到字符串模式。
2. 连续书写多个字符，字符之间保持适当间隔。
3. 系统分割轨迹并显示组合结果。

### 英文单词翻译

1. 启动 PC 接收端。
2. STM32 进入翻译模式。
3. 一个一个写入英文字母。
4. 写错可用删除按钮删除末尾字符。
5. 按发送按钮，PC 端翻译并语音播报，结果返回 STM32。

### PC-CNN 协同识别

1. PC 端安装 PyTorch 并启动 `web_receiver.py`。
2. STM32 进入 PC-CNN 模式。
3. STM32 采集字符图像并发送到 PC。
4. PC 使用 `02_PC端网页与训练工具/训练与模型工具/models/cnn_emnist_byclass.pth` 识别后回传。

## 7. 训练与评估

训练脚本位于：

```text
02_PC端网页与训练工具/训练与模型工具/
```

常用脚本：

- `train_cnn_mnist.py`：训练 MNIST CNN。
- `train_cnn_emnist.py`：训练 EMNIST 数字和字母 CNN。
- `make_testsets.py`：制作 MNIST/个人测试集 BMP。
- `make_emnist_testsets.py`：制作 EMNIST 测试集。
- `evaluate_testsets.py`：评估测试集准确率。
- `evaluate_emnist_testsets.py`：评估 EMNIST 测试集。

原始 MNIST/EMNIST 数据不完整随包发布时，可按脚本参数自行下载或指定数据目录。

## 8. 常见问题

- PC 网页没有数据：确认 STM32 和电脑在同一网络，IP 配置正确，只运行一个 `web_receiver.py`。
- TF 卡挂载失败：确认 TF 卡已格式化为 FAT/FAT32，并且根目录包含 `MODEL/TEST/Font`。
- 中文显示异常：确认 `03_TF卡示例文件/Font/GB2312_H1616.FON` 已复制到 TF 卡 `Font/` 目录。
- 训练脚本缺少 torch/numpy：先执行 `pip install -r requirements.txt`。
