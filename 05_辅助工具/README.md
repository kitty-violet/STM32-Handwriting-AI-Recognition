# 辅助工具说明

本目录放给后来同学复现项目时用的小工具，不参与 STM32 固件编译。

## 工具列表

- `检查开源发布包.ps1`：检查关键文件是否存在、是否还有个人路径/热点/密码/IP、是否有 Keil/Python 生成物。
- `复制TF卡示例文件.ps1`：把 `03_TF卡示例文件/` 一键复制到 TF 卡盘符。

## 使用方式

在 PowerShell 中进入项目根目录后执行，例如：

```powershell
.\05_辅助工具\检查开源发布包.ps1
.\05_辅助工具\复制TF卡示例文件.ps1 -TfRoot X:\
.\05_辅助工具\生成发布压缩包.ps1
```

其中 `X:\` 表示你的 TF 卡盘符，实际使用时改成电脑上显示的盘符即可。
